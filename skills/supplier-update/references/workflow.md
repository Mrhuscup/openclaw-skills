## 住建部爬虫技术细节

### 页面结构

住建部安全生产许可证查询页面 URL：
```
https://zlaq.mohurd.gov.cn/fwmh/bjxcjgl/fwmh/pages/construction_safety/qyaqscxkz/qyaqscxkz
```

### 关键元素

| 元素 | CSS选择器 | 说明 |
|------|----------|------|
| 企业名称输入框 | `#qymc` | placeholder: "请输入完整企业名称" |
| 展开按钮 | `a.btn.show` | 展开搜索面板 |
| 查询按钮 | `a.btn.search` | 触发搜索 |
| 结果表格 | `table` | 包含搜索结果 |
| 有效期列 | `td:nth-child(9)` | 第9列，明文显示 |

### 字体加密问题

页面使用自定义字体加密，以下字段显示为乱码：
- 企业名称
- 统一社会信用代码
- 法定代表人
- 安全生产许可证编号
- 发证机关

**以下字段为明文，可直接提取：**
- 证书状态（有效/无效）
- 有效期结束日期

### API 调用流程

1. 页面加载时调用 `getAqscxkzCount` 获取总数
2. 点击查询后，页面 JS 动态加载数据到表格
3. 数据在表格中直接可见

## 性能考量

### 查询时间

- 单次查询：约 5-10 秒
- 批量查询：建议间隔 2-3 秒
- 100 家公司：约需 10-15 分钟

### 缓存策略

使用内存缓存避免重复查询：
```python
self.query_cache = {}  # {公司名: {'date': 日期, 'source': '住建部官网'}}
```

## 已知限制

1. **字体加密**：无法获取公司名称等加密字段
2. **会员限制**：标招网需要会员账号
3. **网站依赖**：依赖住建部网站稳定性
4. **IP 限制**：频繁访问可能被限制
